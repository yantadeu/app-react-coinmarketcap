Aplicação em React para consumir API Pública CoinMarketCap 
==========================================================

## Principais Tecnologias utilizadas

- React
- Redux
- Bootstrap/ Material
- Yarn


### Instalação

```bash
yarn install / npm install 
```
### bundle dev (*+ redux-devtools*)

```bash
npm run dev
```

### testes

```bash
npm run test
```

### bundle production


```bash
npm run prod
```

### iniciar aplicação

```bash
npm run start
```
