Aplicação em React para consumir API Pública CoinMarketCap 
==========================================================

## Principais Tecnologias utilizadas

- React
- Redux
- Bootstrap


### Instalação

```bash
npm install
```
### bundle dev (*+ redux-devtools*)

```bash
npm run dev
```

### dev (*+ redux-devtools*)

```bash
npm run start
```

### testes

```bash
npm run test
```

### bundle production


```bash
npm run prod
```
